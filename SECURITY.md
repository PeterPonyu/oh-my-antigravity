# Security Policy

oh-my-antigrav is experimental and pre-0.1.0. Please do not treat it as a production
security boundary or a published package.

## Reporting a vulnerability

Open a private security advisory on GitHub if available. If private advisories
are unavailable, contact the repository owner directly and avoid posting exploit
details in a public issue until a fix plan exists.

Include:

- affected commit or branch,
- reproduction steps,
- whether the issue can trigger network access, publication, credential exposure,
  or filesystem writes,
- suggested mitigation if known.

## Supported versions

No released versions are supported yet. Security fixes land on `main` before the
first usable release is cut.

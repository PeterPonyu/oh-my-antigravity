# Code of Conduct

oh-my-antigrav is a small, experimental project. This Code of Conduct keeps it a
respectful place to collaborate. It is original to this repository.

## Our standard

Everyone participating in this project — issues, pull requests, reviews, and
discussions — is expected to:

- Be respectful and assume good faith.
- Focus on the technical substance; critique ideas, not people.
- Welcome newcomers and questions.
- Accept review feedback gracefully and give it constructively.

The following are not acceptable:

- Harassment, insults, or discriminatory language.
- Personal or political attacks.
- Publishing others' private information without consent.
- Sustained disruption of discussion or review.

## Scope

This applies in all project spaces (the repository, issues, pull requests, and
any official channel) and when an individual is representing the project in
public spaces.

## Reporting

Report unacceptable behavior privately to the repository owner. If GitHub private
security advisories are available, you may open one; otherwise contact the owner
directly. Do not post details of an incident in a public issue. See
[SECURITY.md](./SECURITY.md) for the private-contact expectations this project
already follows.

## Enforcement

The maintainer may remove, edit, or reject contributions that violate this Code
of Conduct, and may warn or block contributors for repeated or severe behavior.
Decisions are made in good faith and explained where reasonable.

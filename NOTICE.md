# Notice and Lineage

Antigravity begins as a clean-room local scaffold. It is inspired by observed product routines in Gajae Code and by lessons from local oh-my-codex / oh-my-claudecode harnesses, but this first scaffold does not copy their source code or assets.

Rules before importing any external code or asset:

1. Confirm license and ownership.
2. Record attribution here.
3. Keep copied material isolated in a small PR.
4. Re-run verification and legal-boundary checks.

Current copied code/assets: none.

# Notice and Lineage

oh-my-antigrav begins as a clean-room local scaffold. It is inspired by observed
product routines in Gajae Code and by lessons from local oh-my-codex /
oh-my-claudecode harnesses, but this first scaffold does not copy their source
code or assets.

## Import rules

Before importing any external code, asset, template, or substantial text:

1. Confirm license and ownership.
2. Record attribution in the template below.
3. Keep copied material isolated in a small PR.
4. Re-run verification and legal-boundary checks.

## Attribution template

Use one entry per named upstream before importing material.

```text
Upstream: <project/person/repository>
Material: <file, asset, text, or concept being imported>
License: <SPDX identifier or exact license URL>
Copyright: <copyright holder and year, if known>
Import PR: <PR URL>
Local path: <path in this repository>
Changes: <summary of modifications>
Reason: <why this import is necessary instead of a clean-room rewrite>
```
